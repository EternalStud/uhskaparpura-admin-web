info Instructions

  1. Run your local Python script: python eshikshakosh_scraper.py.
  2. Provide credentials and complete the captcha in the browser that opens.
  3. Wait for the scraping process to finish and generate the Excel output file.
  4. Upload the generated Excel file (typically named Shikshakosh_*.xlsx or Shikshakosh_MultiSchool_*.xlsx from the scraper's output folder) into the upload zone above.
  5. Verify the academic year and student count in the summary section.
  6. Click Sync Database to overwrite the matching session records in Google Sheets database.
